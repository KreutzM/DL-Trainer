# Teacher Prompt: task_card_generation

## Aufgabe
Leite aus dem Quellabschnitt eine Task-Card für genau eine Supportabsicht ab.

## Regeln
- Maximal eine Kernabsicht pro Task-Card
- Nur belegte Fakten
- Keine impliziten Annahmen
- Versionen und Voraussetzungen nennen, wenn relevant

## Ausgabeformat
JSON gemäß `schemas/task_card.schema.json`
