# Skill: task-card-generator

Nutze diesen Skill, wenn aus normalisierten Handbüchern lösungsorientierte Support-Task-Cards abgeleitet werden sollen.

## Ziel
Eine Task-Card beantwortet genau eine Supportabsicht:
- Aktivieren/Deaktivieren
- Fehler beheben
- Einstellung finden
- Bedeutung einer Meldung erklären
- Navigation in UI

## Regeln
- Nur faktenbasiert aus Quelltext
- Immer Provenance
- Versionen getrennt halten
- Schrittfolgen nur übernehmen, wenn sie explizit im Handbuch stehen
