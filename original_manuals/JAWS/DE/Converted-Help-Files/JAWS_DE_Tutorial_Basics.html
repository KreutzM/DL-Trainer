Über JAWS

Über JAWS

Willkommen in der Welt von JAWS!

Sie werden nun die Stärke und Leistungsfähigkeit eines Bildschirmleseprogramms kennenlernen, das blinden und sehbehinderten Computeranwendern den Zugriff auf die zur Zeit gängigsten Computerprogramme und das Internet ermöglicht. JAWS ist das leistungsstärkste Bildschirmleseprogramm für Microsoft Windows.

In diesem speziell für Neueinsteiger konzipierten Kapitel werden wir Ihnen die Grundkenntnisse für den Einstieg in JAWS vermitteln. In nur wenigen Schritten zeigen wir Ihnen, wie Sie auf die Hilfe zugreifen, die grundlegenden Befehle zum Navigieren auf der Windows Umgebung verwenden, JAWS Spracheinstellungen nach Ihrer Wahl vornehmen und alle Computerprogramme, die auf Ihrem System installiert sind starten können. JAWS ermöglicht den Zugriff auf fast alle heute im Internet vorhandenen Informationen und ausbildungs- und arbeitsplatzspezifischen Anwendungen.

Am Ende eines jeden Themas können Sie fortfahren, indem Sie den Link Nächstes Thema aktivieren.

Über die JAWS Cursor

Über die JAWS Cursor

Beim Navigieren auf der Windows Oberfläche setzt JAWS unterschiedliche Cursor ein. Diese Cursor bestimmen den Fokus für die JAWS Sprachausgabe und sind häufig direkt mit den Windows Cursor verbunden.

Der PC Cursor - Folgt den Tastatureingaben

Der PC Cursor ist direkt mit windowsspezifischen und anwendungsbezogenen Tastaturfunktionen verbunden. Dieser Cursor wird eingesetzt, wenn man Informationen eingibt, sich durch die Optionen in einem Dialogfeld bewegt und eine Option oder ein Symbol markiert. Bei der Eingabe von Informationen folgt der PC Cursor jeder Taste, die Sie drücken. Wenn Sie in einem Menü oder in einem Dialogfenster eine Auswahl treffen, hebt der PC Cursor das aktuell ausgewählte Objekt hervor. Der PC Cursor wird beim Starten von JAWS standardmäßig aktiviert. Wenn Sie einen anderen Cursor verwenden, schalten Sie mit NUM PLUS den PC Cursor wieder ein.

Der JAWS Cursor - Folgt dem Mauszeiger

Der JAWS Cursor ist mit den Mauszeigerfunktionen in Windows und in anderen Anwendungen verbunden. Er wird dazu verwendet, die Art von Informationen zu lesen, die der PC Cursor nicht lesen kann, wie zum Beispiel Informationen in der Werkzeugleiste. Der Mauszeiger folgt dem JAWS Cursor, wenn dieser bewegt wird, so dass Sie Zugriff auf Informationen in einem Anwendungsfenster erhalten, die außerhalb der Reichweite des PC Cursors liegen. Sie aktivieren den JAWS Cursor mit NUM GEDANKENSTRICH.

Der unsichtbare Cursor - Erkunden, ohne den Fokus zu bewegen

Mit dem Unsichtbaren Cursor bewegen Sie sich auf dem Bildschirm, ohne die Position des PC Cursors oder Mauszeigers zu verändern. Mit diesem Cursor können Sie Text lesen und auf Optionen zugreifen, die Sie mit dem PC Cursor nicht erreichen. Der unsichtbare Cursor kann auch dazu verwendet werden, sich einen schnellen Überblick über die momentan am Bildschirm angezeigten Informationen zu verschaffen. Sie aktivieren den unsichtbaren Cursor mit NUM MINUS zweimal hintereinander.

Die Touch Cursor - Objekt Navigation

Der Touch Cursor ermöglicht das Navigieren in Windows Umgebungen auf einem Touchscreen, den Sie auf vielen Tablet-Computern finden, die unter Windows 8 oder höher laufen. Um JAWS auf einen Touchscreen zu steuern, werden Sie einen oder mehrere Finger nutzen müssen, um verschiedene Gesten direkt auf dem Bildschirm auszuführen, wie tippen, streichen oder wischen. Wenn über die Tastatur aktiviert, dann können Sie Kurztasten zum Simulieren von Touchscreen-Gesten verwenden, um sich zwischen den Objekten einer Anwendung zu bewegen, um auf Informationen zuzugreifen, die über den JAWS oder PC Cursor nicht lesbar zugänglich wären.

Der Touch Cursor steht immer zur Verfügung, wenn auf einem Touchscreen navigiert wird. Um den Touch Cursor über die Tastatur zu aktivieren, drücken Sie UMSCHALT+NUM PLUS.

Der Virtuelle PC Cursor - HTML Umgebung

Der Virtuelle PC Cursor arbeitet weitgehend so wie der PC Cursor, aber er wird standardmäßig aktiviert, wenn man auf ein HTML Dokument trifft, wie etwa eine Webseite im Internet. Der virtuelle PC Cursor sagt die Anzahl der Elemente auf der Seite an, sobald sich diese erstmals öffnet. Zum Beispiel die Anzahl der Links, Überschriften und Regionen. Sie können die PFEILTASTEN zum Navigieren und Lesen von Dokumenten verwenden, oder Sie nutzen die Schnellnavigationstasten, um auf spezielle Elemente, wie etwa Absätze, Tabellen oder Überschriften zu springen.

So bekommen Sie Hilfe

So bekommen Sie Hilfe

JAWS bietet Hilfe und Informationen auf verschiedene Weise und an unterschiedlichen Orten an. In diesem Kapitel werden die wichtigsten Befehle aufgelistet, mit denen Sie schnelle Hilfe für JAWS, Windows oder die aktuell laufende Anwendung erhalten können.

  • JAWS TASTE+1 für die Tastaturhilfe
  • JAWS TASTE+H für die Kurztastenhilfe
  • JAWS TASTE+F1 für die Bildschirmabhängige Hilfe
  • JAWS TASTE+F1, F1 für JAWS Hilfe in Anwendungen
  • JAWS TASTE+J, ALT+H  für das JAWS Hilfe Menü

Überblick über die Lese-Funktionen

Überblick über die Lese-Funktionen

Für umfangreichere Informationen, wie Sie mit JAWS lesen können, lesen Sie bitte das Thema Den Bildschirm und Dokumente lesen und das Thema Erweitertes Lesen von Bildschirm und Dokumenten.

Nutzen Sie die folgenden Tasten auf dem Nummernblock (NUM PAD), um mit dem Lesen in JAWS zu starten.

  • NUM 5 : Zeichen lesen.
  • JAWS Taste+NUM 5: Wort lesen.
  • JAWS TASTE+NUM 5 zweimal: Wort buchstabieren.
  • JAWS TASTE+PFEIL LINKS: Vorheriges Wort lesen.
  • JAWS TASTE+PFEIL RECHTS: Nächstes Wort lesen.
  • JAWS TASTE+PFEIL RAUF: Zeile lesen.
  • JAWS TASTE+POS1 (NUM 7): Zeile links vom Cursor lesen.
  • JAWS TASTE+SEITE RAUF (NUM 9): Zeile rechts vom Cursor lesen.
  • JAWS TASTE+SEITE RUNTER (NUM 3): Unterste Zeile des Fensters lesen.
  • JAWS TASTE+ENDE (NUM 1): Oberste Zeile des Fensters lesen.
  • STRG+JAWS TASTE+PFEIL RUNTER Querlesen starten
  • JAWS TASTE+ALT+W Fenster virtualisieren
  • EINFÜGEN+UMSCHALT+V Aktuelles Element virtualisieren
  • JAWS TASTE+STRG+W Wort-Index

Starten Sie JAWS

Starten Sie JAWS

JAWS manuell starten

Um JAWS manuell zu starten, gehen Sie bitte folgendermaßen vor:

  • Drücken Sie WINDOWS Taste+R , um das Dialogfeld Ausführen zu öffnen, schreiben Sie JAWSX, wobei das X für die JAWS Versionsnummer steht und drücken Sie EINGABE.

JAWS automatisch starten

Wenn Sie JAWS zum ersten Mal installieren, dann erscheint der JAWS Start-Assistent. Das Dialogfeld JAWS Einstellungen anpassen enthält mehrere Optionen zum Start von JAWS. Die Standardeinstellungen gewährleisten, dass JAWS immer geladen und gestartet wird und läuft, bevor und nachdem Windows gestartet wurde. Werden diese Einstellungen verändert, und Sie möchten später die Standard-Einstellungen wiederherstellen, dann gehen Sie bitte folgendermaßen vor, damit JAWS automatisch startet:

  1. Drücken Sie JAWS TASTE+J, um das JAWS Programmfenster zu öffnen.
  2. Wählen Sie im Menü Optionen den Eintrag Grundeinstellungen.
  3. Wählen Sie den Schalter JAWS automatisch starten. Der Dialog Optionen für den JAWS Start erscheint. Verfügen Sie über Administratorrechte, dann erscheinen einige Optionen in dem Dialogfenster. Verfügen Sie über ein Standard-Anwender-Konto, dann erscheint nur ein Element.
  4. Für einen Administrator-Konto gehen Sie wie folgt vor:
  • Aktivieren Sie das Kontrollfeld JAWS im Anmeldedialog starten.
  • Deaktivieren Sie das Kontrollfeld JAWS nach dem Anmelden für alle Anwender starten.
  • Wählen Sie in der Ausklappliste JAWS nach dem Anmelden für diesen Anwender starten den Eintrag Immer.

Für ein Standardbenutzer-Konto gehen Sie wie folgt vor:

  • Wählen Sie in der Ausklappliste JAWS nach dem Anmelden für diesen Anwender starten den Eintrag Immer.
  1. Wählen Sie OK, um die Änderungen zu speichern und das Dialogfeld JAWS Start zu schließen.
  2. Als nächstes schließen Sie das Dialogfenster Grundeinstellungen. Beim nächsten Windowsstart wird JAWS nun automatisch gestartet und ist eines der zuerst verfügbaren Programme.

JAWS beenden

  • Drücken Sie EINFÜGEN+F4, um JAWS zu beenden.

Stimmeneinstellungen

Stimmeneinstellungen

Die JAWS Stimmeneinstellungen für die Sprachausgabe lassen sich unterschiedlich anpassen, so dass Sie die einzelnen JAWS Funktionen akustisch unterscheiden können. So sollte zum Beispiel die Stimme für den PC Cursor anders eingestellt sein, als die Stimme für das Tastaturecho oder für die Großbuchstabenansage.

Gehen Sie bitte folgendermaßen vor, wenn Sie Stimmeneinstellungen öffnen möchten:

  1. Drücken Sie EINFÜGEN+J, um das JAWS Programmfenster zu öffnen.
  2. Drücken Sie ALT+O, um das Menü Optionen zu öffnen.
  3. Drücken Sie T und dann EINGABE, um das Dialogfeld Stimmeneinstellungen zu öffnen. Hier können Sie ein bestehendes Profil oder eine Standard-Stimmenvorlage auswählen und verschiedene Einstellungen, wie Sprache, Stimme, Sprechgeschwindigkeit, Satzzeichenansage und anderes nach Ihren Wünschen konfigurieren.
  4. Nach dem Ändern der Einstellungen, speichern Sie das Stimmenprofil und schließen Sie das Dialogfenster Stimmeneinstellungen.

Um eine Stimmenprofil auszuwählen, gehen Sie bitte folgendermaßen vor:

  1. Drücken Sie JAWS TASTE+STRG+S, um das Dialogfenster Stimmenprofil auswählen zu öffnen.
  2. Nutzen Sie die Ausklappliste Standard, um eine Stimmenprofil auszuwählen, dass von JAWS global verwendet wird.
  3. Nutzen Sie die Ausklappliste Für, um ein Stimmenprofil auszuwählen, dass nur für die aktuell geöffnete Anwendung verwendet wird. Wählen Sie Standard, dann verwendet JAWS das aktuell in der Standard Ausklappliste gewählte Profil.

Die Systemsteuerung

Die Systemsteuerung

Die Systemsteuerung ist eine Komponente, in der Sie die Windows Einstellungen ändern können. Sie sollten sich etwas Zeit nehmen, um die Systemsteuerung und die vielen Funktionen und Elemente, die sich dort befinden, zu erkunden. Um die Systemsteuerung zu öffnen, wählen Sie Start, Einstellungen, Systemsteuerung.

Standardmäßig ist die Systemsteuerung als Liste von Symbolen, die in Zeilen und Spalten angeordnet sind, aufgebaut. Drücken Sie die LEERTASTE, um das erste Element hervorzuheben, und navigieren Sie mit den PFEILTASTEN durch die Auswahlmöglichkeiten.

Einige Elemente der Systemsteuerung, die für Sie interessant sein könnten, sind:

  • Datum und Uhrzeit – Hier stellt man das Datum, die Uhrzeit und die Zeitzone ein. Drücken Sie JAWS TASTE+F12, um die Uhrzeit zu hören, und JAWS TASTE+F12 zweimal schnell hintereinander, um das Datum zu hören.
  • Anzeige – Ändert die Desktopdarstellung, wie zum Beispiel den Hintergrund, Bildschirmschoner, Farben, Schriftgrade und Bildschirmauflösung.
  • Sounds und Audiogeräte – Ändert das Soundschema des Computers oder konfiguriert die Einstellungen für die Lautsprecher und Aufnahmegeräte.
  • Multimedia - Ermöglicht Einstellungen zu Geräten, die für Multimedia genutzt werden, inklusive Video, MIDI, etc.
  • Software – Bietet Ihnen einen Weg nachzusehen, welche Anwendungen auf Ihrem Computer installiert sind, darüber hinaus können Sie Programme und Windows-Komponenten installieren oder entfernen.
  • Eingabehilfen – Passt die Computereinstellungen Ihrer Sehstärke, Ihrem Gehör und Ihrer Beweglichkeit entsprechend an. Dies beinhaltet die Einrastfunktion der Tastatur und Hoher Farbkontrast bei der Darstellung.

Hinweis:Änderungen in der Systemsteuerung können die Sprachausgabe von JAWS beeinträchtigen. Gehen Sie beim Ändern dieser Einstellungen bitte mit Vorsicht zu Werke.

Orientierung auf dem Desktop

Orientierung auf dem Desktop

Stellen Sie sich Windows wie einen traditionellen Büroarbeitsplatz auf Ihrem Computer vor. In Ihrem Büro, egal ob am Arbeitsplatz oder zu Hause, befindet sich höchstwahrscheinlich ein Schreibtisch. Es gibt auch verschiedene Aktenordner, die Unterlagen (Dateien) enthalten.

Windows ist genauso strukturiert, nur dass es elektronisch ist. Der erste Bildschirm, mit dem Sie nach dem Start Ihres Computers konfrontiert werden, ist der Windows Desktop. Auf dem Desktop befinden sich alle Werkzeuge, die Sie zur Erledigung Ihrer Arbeit benötigen. Es handelt sich bei den Werkzeugen um Softwareanwendungen, also Computerprogramme, mit denen Sie Ihre Arbeit erledigen können. Außerdem finden Sie auf dem Desktop Ordner, die Ihre Dateien (Akten) enthalten. Dies können fertige oder noch in Arbeit befindliche Dokumente sein.

Stellen Sie sich den Desktop als die Eltern aller Fenster vor. Stellen Sie sich alle Fenster wie Anwendungen oder Dateien als untergeordnete oder Kindfenster auf dem Desktop vor.

Um von einer Komponente des Desktops zur nächsten zu gelangen, drücken Sie TAB. Um sich rückwärts zu bewegen, drücken Sie UMSCHALT+TAB. Sie werden feststellen, dass die Komponenten des Desktops kreisförmig angeordnet sind. Falls Sie mit TAB nicht zwischen Komponenten navigieren können, drücken Sie WINDOWSTASTE+M, um alle Anwendungen zu minimieren und zur Desktop Listenansicht zurückzukehren. Sobald Sie sich auf einer bestimmten Desktop Komponente wie zum Beispiel der Taskleiste befinden, benutzen Sie die PFEILTASTEN, um sich von einem Element zum nächsten zu bewegen.

Desktop Listenansicht

Die Desktop Listenansicht enthält eine Liste von Symbolen oder Verknüpfungen, aus der heraus Sie Anwendungen starten können. In dieser Liste können Sie aber auch auf Ordner und/oder Verzeichnisse zugreifen, die Sie auf dem Windows Desktop abgelegt haben.

Um auf dem Windows Desktop zu navigieren, müssen Sie zuerst die LEERTASTE drücken, um das erste Element in der Listenansicht zu markieren, dabei handelt es sich bei Windows XP um Eigene Dateien. Nachdem Sie die LEERTASTE gedrückt haben, können Sie sich mit den Pfeiltasten durch die Spalten und Reihen der Desktop Symbole bewegen. Drücken Sie EINGABE auf dem gewünschten Symbol, um es zu öffnen.

Der Desktop Hintergrund ist in der Regel einfarbig. Die standardmäßige Farbeinstellung können Sie verändern, ein Hintergrundmuster auswählen oder ein Bild auf dem Desktop speichern. Weitere Informationen darüber, wie Sie diese Einstellungen verändern können, finden in der Windows Hilfe.

Drücken Sie TAB, um auf die nächste Desktop Komponente, den Start Schalter, zu springen.

Start Schalter

Mit TAB springen Sie auf den Start Schalter, JAWS sagt dann "Start Schalter". Sie finden ihn in der linken unteren Bildschirmecke. Drücken Sie EINGABE, um das Startmenü zu öffnen. Der erste Menüeintrag, den Sie hören, steht am Anfang der Liste. Wenn Sie das Startmenü nicht verändert haben, öffnet es sich auf Programme, und Programme Untermenü wird angesagt. Die Aussage "Untermenü" deutet darauf hin, dass Sie mit PFEIL RECHTS oder EINGABE ein weiteres Menü öffnen können, das sich innerhalb des Hauptmenüs befindet.

Mit dem Start Schalter greifen Sie neben anderen Windows Funktionen auf all Ihre Programme zu. Die Einträge des Startmenüs lauten in der Regel Programme, Dokumente, Einstellungen, Suchen, Hilfe und Support, Ausführen und Herunterfahren. Alternativ können Sie das Startmenü auch öffnen, indem Sie auf die WINDOWSTASTE drücken, eine befindet sich an zweiter Stelle von links in der untersten Reihe Ihrer Windows Tastatur, die andere an dritter Stelle von rechts.

Sie können das Startmenü auch anpassen, indem Sie weitere Verknüpfungen wie zum Beispiel JAWS hinzufügen. Um Anpassungen vorzunehmen, gehen Sie auf den Eintrag Einstellungen im Startmenü, und drücken Sie EINGABE. Drücken Sie PFEIL RUNTER, um auf den Eintrag Taskleiste & Startmenü zu springen, danach drücken Sie EINGABE. Mit STRG+TAB springen Sie auf die Registerkarte Startmenü. Aktivieren Sie dort den Auswahlschalter Klassisches Startmenü und aktivieren Sie dann den Schalter Anpassen. Im Dialogfeld Klassisches Startmenü anpassen aktivieren Sie den Schalter Hinzufügen. Dann geben Sie den Pfad für die Verknüpfung ein, die Sie hinzufügen möchten, oder drücken auf den Schalter Durchsuchen, um danach zu suchen. So finden Sie zum Beispiel JAWS unter C:\Programme\Freedom Scientific\JAWS\X (wobei X die JAWS Versionsnummer ist)\JFW.exe. Als nächstes wählen Sie aus, wo die Verknüpfung erscheinen soll. Wenn die neue Verknüpfung im Hauptmenü erscheinen soll, übernehmen Sie den voreingestellten Ordner Programme. Danach müssen Sie die neue Verknüpfung benennen. Sie entfernen eine Verknüpfung, indem Sie auf den Schalter Entfernen drücken anstatt auf den Schalter Hinzufügen.

Wenn Sie Hilfe zur Navigation in Dialogfeldern benötigen, lesen Sie bitte das Kapitel Beispiel Dialogfelder.

Drücken Sie ESCAPE, um das Startmenü zu schließen und auf den Start Schalter zurückzukehren. Mit TAB springen Sie auf die Schnellstartleiste. Sie hören nicht "Schnellstartleiste", sondern JAWS sagt statt dessen "Werkzeugleiste".

Schnellstartleiste

Die Schnellstartleiste befindet sich direkt rechts neben dem Start Schalter am unteren Bildschirmrand. Wenn Sie zur Schnellstartleiste springen, sagt JAWS "Symbolleiste". Sobald Sie JAWS "Symbolleiste" sagen hören, drücken Sie die Anwendungstaste, und dann PFEIL RUNTER, um Symbolleisten auszuwählen. Drücken Sie EINGABE und nutzen Sie PFEIL RUNTER, um zur Schnellstartleiste zu gelangen und drücken Sie EINGABE. Wenn Sie jetzt mit der TAB-Taste über die Komponenten des Windows Desktops springen, dann werden Sie "Symbolleiste" nicht mehr hören.

Mit TAB springen Sie in die Taskleiste.

Taskleiste

Die Taskleiste befindet sich rechts neben der Schnellstartleiste. Ist die Schnellstartleiste ausgeblendet, dann befindet sich die Taskleiste rechts neben dem Start Schalter. In der Taskleiste werden alle zur Zeit laufenden Anwendungen aufgeführt. Nachdem Sie mit TAB auf die Taskleiste gesprungen sind, drücken Sie PFEIL RECHTS und LINKS, um sich durch die laufenden Anwendungen zu bewegen. Drücken Sie EINGABE auf der gewünschten Anwendung, damit diese den Fokus erhält. Wenn nur ein Element in der Taskleiste angesagt wird, ist nur eine laufende Anwendung vorhanden.

Fensterliste

Mit JAWS TASTE+F10 öffnen Sie ein Dialogfeld, in dem alle aktuell laufenden Anwendungen aufgelistet werden. Es handelt sich bei dieser Liste um die gleichen Anwendungen, die in der Taskleiste zu finden sind. Für weitere Informationen lesen Sie Dialogfeld Fensterliste.

Infobereich

Die letzte Komponente auf dem Windows Desktop ist der Infobereich. Sie finden ihn in der rechten unteren Bildschirmecke. Allerdings können Sie mit TAB diesen Bereich nicht erreichen. Ohne JAWS können Sie auf den Infobereich nur mit der Maus zugreifen. Mit JAWS TASTE+F11 greifen Sie jedoch schnell auf die Elemente im Infobereich zu. Dieser Befehl öffnet ein Dialogfeld, das alle laufenden Anwendungen im Infobereich auflistet. Weitere Information darüber, wie man mit JAWS auf den Infobereich zugreifen kann, finden Sie unter Ein Symbol des Infobereichs auswählen.

Im Infobereich finden Sie auch die aktuelle Uhrzeit und das Datum. Drücken Sie JAWS TASTE+F12, um die Uhrzeit zu hören, und JAWS TASTE+F12 zweimal schnell hintereinander, um das Datum zu hören.

Eine Kurztaste zuweisen

Mit Kurztasten kann man sehr schnell Programme wie zum Beispiel JAWS starten. Folgen Sie den untenstehenden Anweisungen, um JAWS eine Kurztaste zuzuweisen.

  1. Drücken Sie WINDOWSTASTE+M, um alle Anwendungen zu minimieren und den Fokus auf dem Desktop zu platzieren.
  2. Drücken Sie J, den Anfangsbuchstaben der Desktop Verknüpfung JAWS. Es kann sein, dass Sie diesen Buchstaben mehrmals drücken müssen, bis Sie "JAWS" hören.
  3. Mit ALT+EINGABE öffnen Sie das Dialogfeld Eigenschaften. Überprüfen Sie, ob Sie sich auf der Registerkarte Verknüpfung befinden.
  4. Drücken Sie ALT+T oder TAB, um in das Eingabefeld Tastenkombination zu springen. Drücken Sie J, um diesen Buchstaben als Kurztaste für JAWS zuzuweisen.
  5. Drücken Sie JAWS TASTE+PFEIL RAUF, um die von Ihnen gewählte Kurztaste zu hören.

Hinweis: Wenn Sie eine Tastenkombination eingeben, die bereits zugewiesen ist, fügt Windows automatisch eine weitere Taste Ihrer Kombination hinzu. Verwenden Sie den Befehl JAWS TASTE+PFEIL RAUF, um Ihre Kurztaste zu überprüfen. Wenn Sie zum Beispiel ein J für JAWS eingegeben haben, fügt Windows STRG+ALT dem J hinzu, damit diese Tastenkombination auch wirklich funktioniert.

  1. Mit EINGABE wird die Kurztaste aktiviert.

Hinweis: Falls diese Option im Dialogfeld Eigenschaften nicht vorhanden ist, wurde ein Desktop Objekt markiert und keine Desktop Verknüpfung, so dass Sie ein anderes Element auswählen müssen.

Der Papierkorb

Der Papierkorb

Wenn Sie eine Datei oder einen Ordner löschen, werden diese Elemente in den Windows Papierkorb verschoben. Der Papierkorb ist ein Behälter, in dem gelöschte Objekte zwischengelagert werden, bevor man sie dauerhaft vom System entfernt. Entweder öffnen Sie den Papierkorb mit dem Grafiksymbol auf dem Windows Desktop oder aus dem Windows Explorer heraus. Im Dialogfenster Papierkorb wird eine Listenansicht geöffnet. Sie können Objekte aus dem Papierkorb an ihrem ursprünglichen Ort wiederherstellen. Markieren Sie ein Objekt, das Sie wiederherstellen möchten und drücken Sie dann auf die Windows APPLIKATIONSTASTE; das ist die zweite Taste von rechts auf der untersten Tastaturreihe. Mit dieser Taste werden Kontextmenüs geöffnet. Drücken Sie PFEIL RUNTER, und gehen Sie auf den ersten Menüeintrag Wiederherstellen. Mit EINGABE aktivieren Sie die Funktion. Sie können ein Objekt auch wiederherstellen, indem Sie den Eintrag Wiederherstellen aus dem Dateimenü aufrufen.

Von Zeit zu Zeit sollten Sie Ihren Papierkorb leeren. Öffnen Sie den Papierkorb und wählen Sie Papierkorb leeren aus dem Dateimenü. Damit werden alle Elemente, die sich im Papierkorb befinden, unwiderruflich gelöscht.

Beispieldialoge

Beispieldialoge

In einem Dialogfenster können unterschiedliche Steuerelemente vorhanden sein. HJPad ist eine Anwendung, die von Freedom Scientific entwickelt wurde, damit Sie sich in einer sicheren Umgebung mit der Navigation in Dialogfenstern vertraut machen können. Die Beispieldialoge finden Sie im Programm HJPad im Menü Extras. Verwenden Sie diese Dialoge als eine sichere Umgebung, in der Sie die Navigation von unterschiedlichen Dialogfenstern und den darin enthaltenen Steuerelementen üben können.

Die ersten Steuerelemente, auf die Sie in unserem Beispieldialog stoßen, sind eine Reihe von Auswahlschaltern. Auswahlschalter erscheinen als kleine Kreise, die sich normalerweise links neben den Auswahlen befinden, aufgereiht in einer Liste. Verwenden Sie die Taste PFEIL RUNTER, um sich durch die Liste der Auswahlschalter zu bewegen. Wenn Sie die für Sie richtige Auswahl gefunden haben, drücken Sie TAB, um auf das nächste Steuerelement zu springen. In unserem Beispieldialog sollen Sie eine Anredeform auswählen: Herr, Frau oder Fräulein.

Hinweis: Es kann nur ein Auswahlschalter zur Zeit in einer Gruppe ausgewählt werden.

Eingabefelder

Wenn JAWS sagt: "Vorname Doppelpunkt Eingabefeld", befinden Sie sich in einem Eingabefeld. Eingabefelder verhalten sich ähnlich wie Eingabefelder in Textverarbeitungsprogrammen. Sie können hier Text eingeben und die normalen Navigationsbefehle verwenden, die Sie aus der Textverarbeitung kennen. Geben Sie hier Ihren/einen Vornamen ein, und drücken Sie TAB, um zum nächsten Steuerelement zu gelangen, bei dem es sich auch um ein Eingabefeld handelt. Geben Sie nun Ihren/einen Nachnamen ein und drücken Sie TAB, um zum nächsten Eingabefeld zu gelangen, bei dem es sich um ein mehrzeiliges Eingabefeld handelt. In ein mehrzeiliges Eingabefeld können Sie mehr als eine Zeile Text eingeben. Sie sollten hier Ihren/einen Straßennamen eingeben. Bei Bedarf fügen Sie mit EINGABE eine weitere Zeile hinzu. Drücken Sie TAB, um auf das nächste Steuerelement zu springen.

Kombiniertes Eingabefeld

Das nächste Steuerelement ist ein kombiniertes Eingabefeld. Hierbei handelt es sich eigentlich um die Kombination zweier Elemente: ein Eingabefeld und ein Listenfeld. Sie können entweder einen Wert eingeben oder ALT+PFEIL RUNTER drücken, um ein Listenfeld zu öffnen, welches Vorschläge für den Inhalt dieses Felds enthält. Gehen Sie mit PFEIL RAUF und PFEIL RUNTER durch die Liste, und schließen Sie die Liste mit ALT+PFEIL RAUF oder ESCAPE. Wenn Sie in der Vorschlagsliste keinen passenden Eintrag gefunden haben, geben Sie einen Wert in das Eingabefeld ein.

Ausklappliste

Das nächste Feld ist eine einfache Ausklappliste, die sich von einem kombinierten Eingabefeld darin unterscheidet, dass Sie hier keinen Wert oder Text eingeben können. Sie können Ihre Auswahl nur unter den angebotenen Elementen treffen. In unserem Beispiel müssen Sie einen US-Bundesstaat auswählen, in dem Sie wohnen beziehungsweise wohnen möchten. Drücken Sie PFEIL RUNTER, um sich durch die Liste zu bewegen, oder drücken Sie den Anfangsbuchstaben des gewünschten Eintrags. Zum Beispiel F für Florida.

Drehfeld

Beim nächsten Steuerelement handelt es sich um ein Drehfeld, das auch aus zwei Steuerelementen kombiniert ist. Sie können einen Wert in dieses Feld eingeben oder einen Vorschlag mit den Pfeiltasten auswählen. Drehfelder werden vorwiegend für numerische Eingaben verwendet.

Links/Rechts Schieber

Der Links/Rechts-Schieber ist horizontal aufgebaut, so dass Sie mit PFEIL RECHTS und PFEIL LINKS in Ein-Prozent-Schritten wandern können. Wenn Sie größere Schritte machen möchten, verwenden Sie SEITE RAUF und SEITE RUNTER.

Kontrollfeld

Das nächste Steuerelement in unserem Beispieldialog ist ein Kontrollfeld. Ein Kontrollfeld kann aktiviert oder deaktiviert werden. JAWS sagt den aktuellen Status an, wenn man beim Navigieren ein Kontrollfeld erreicht. Mit der LEERTASTE werden Kontrollfelder aktiviert oder deaktiviert. Liegt eine Liste von Optionen vor, denen die Kontrollfelder zugeordnet sind, können diese mit den PFEILTASTEN navigiert werden. Verwenden Sie den Befehl Zeile lesen (JAWS TASTE+PFEIL RAUF), um festzustellen, ob ein Kontrollfeld aktiviert oder deaktiviert ist.

Andere Steuerelemente

Bei dem nächsten Steuerelement in unserem Beispieldialog handelt es sich um den Schalter Formular löschen. Drücken Sie EINGABE auf diesem Schalter, um alle Einträge zu löschen und alle anderen Optionen auf ihre Standardwerte zurückzusetzen.

Der OK Schalter und der Schalter Abbrechen sind die letzten Elemente in unserem Beispieldialog. Sie finden diese beiden Schalter in fast allen Dialogfenstern wieder. OK speichert Ihre Änderungen und wendet diese sofort an, Abbrechen schließt das Dialogfenster, ohne dass von Ihnen vorgenommene Änderungen oder Einträge gespeichert werden. Sobald Sie mit der Bearbeitung eines Dialogfeldes fertig sind, können Sie JAWS TASTE+E drücken, um den Standardschalter des Dialogfensters zu ermitteln. Handelt es sich um den OK Schalter, aktivieren Sie diesen Schalter mit EINGABE.

Mehrseitiges Layout

Das Dialogfenster Mehrseitig unterscheidet sich vom Dialogfenster Einseitig 1 dadurch, dass es aus mehreren Seiten beziehungsweise Registerkarten besteht. Wenn Sie ein Dialogfenster, mit dem Sie nicht vertraut sind, öffnen, hören Sie auf die Worte Registerkarte und Seite, die anzeigen, dass es sich um ein mehrseitiges Dialogfenster handelt. Sobald Sie mit der Bearbeitung der ersten Registerkarte fertig sind, drücken Sie STRG+TAB, um die nächste Registerkarte in den Vordergrund zu holen. STRG+TAB bringt Sie von einer Registerkarte auf die nächste, um schließlich wieder auf der ersten zu landen oder auf der letzten Seite stehenzubleiben.

Listenansicht

Die Listenansicht mit dem Titel Listenansicht für meinen Fortschritt enthält 107 Einträge, die mit den Einträgen des mehrseitigen Dialogfensters übereinstimmen. Wenn Sie also in der Listenansicht eine Änderung vornehmen, wird das entsprechende Kontrollfeld im mehrseitigen Dialogfenster mit umgeschaltet. Um die vorhandenen Einträge der Listenansicht zu hören, drücken Sie PFEIL RAUF und PFEIL RUNTER. In unserem Beispieldialog sagt JAWS an, ob die jeweilige Aufgabe fertig oder noch nicht fertig ist. Drücken Sie C, um ein Element als Fertig zu markieren, und N, um es als Nicht Fertig zu markieren.

Die Einträge in dieser Listenansicht sind in drei Spalten angeordnet. Um die Spaltenüberschriften zu lesen, ziehen Sie den JAWS Cursor zum PC Cursor (JAWS TASTE+NUM MINUS) und drücken PFEIL RAUF. Wenn Sie sich durch die Liste bewegen, liest JAWS nur das, was auf dem Bildschirm erscheint. Manchmal passen Einträge physikalisch nicht in die Spalte. Um die Spalte auszudehnen, drücken Sie STRG+NUM PLUS. Das Ausdehnen der Spalte formatiert die Ansicht neu, so dass der gesamte Spalteninhalt sichtbar wird. Das Ausdehnen der Spalte ist eine temporäre Einstellung und muss jedes Mal, wenn Sie ein Dialogfenster öffnen, erneut durchgeführt werden. Weitere Informationen darüber finden Sie unter Spalten ausdehnen.

Sie können den Status von mehr als einem Eintrag in der Listenansicht gleichzeitig ändern. Um dies durchzuführen, wählen Sie alle Einträge, die Sie verändern möchten, und drücken dann C oder N. Es lassen sich sowohl zusammenhängende als auch nicht zusammenhängende Einträge auswählen. Um zu erfahren, wie Sie Einträge auswählen, lesen Sie bitte Elemente auswählen.

Strukturansicht

In einer Strukturansicht werden Elemente hierarchisch angeordnet und vertikal am Bildschirm dargestellt. Mit PFEIL RAUF und PFEIL RUNTER bewegen Sie sich durch eine Strukturansicht. Wenn Sie "Ebene 0 geschlossen" hören, öffnen Sie mit PFEIL RECHTS diese Ebene. Es erscheinen dann weitere Informationen unterhalb dieser Ebene. Mit PFEIL RUNTER bewegen Sie sich durch die Elemente der nächsten Ebene. Auch hier kann ein Element geschlossen sein, so dass Sie wieder mit PFEIL RECHTS diese Ebene öffnen können. Sie schließen eine Ebene mit PFEIL LINKS oder RÜCKSCHRITT.

Hinweis:JAWS sagt die Positionsnummer der Ebene nur dann an, wenn sie sich ändert.

Um auf das erste Element in einer Strukturansicht zu springen, drücken Sie POS1, mit ENDE springen Sie auf das letzte Element.

Rechts neben der Strukturansicht befindet sich ein zweiter Bereich, in dem Informationen über das in der Strukturansicht gewählte Element angezeigt werden. Mit TAB springen Sie in den rechten Bereich der Strukturansicht. Mit dem Befehl Alles Lesen, JAWS TASTE+PFEIL RUNTER, werden diese Informationen vorgelesen. Mit UMSCHALT+TAB springen Sie in den ursprünglichen Bereich der Strukturansicht zurück.

Der typische Aufbau eines Anwendungsfensters

Der typische Aufbau eines Anwendungsfensters

Dieser Abschnitt beschreibt den Aufbau eines typischen Anwendungsfensters, beginnend am oberen und endend am unteren Rand des Fensters. HJPad, ein einfaches Textverarbeitungsprogramm, das zusammen mit JAWS ausgeliefert wird, dient als Beispielanwendung. Öffnen Sie das Startmenü und wählen Sie Programme, dann den Menüeintrag JAWS. Gehen Sie dann mit PFEIL RUNTER durch das Menü auf den Eintrag HJPad starten. Drücken Sie EINGABE, um die Anwendung zu starten.

Titelzeile

Die Titelzeile befindet sich am oberen Rand des Fensters und zeigt den Dateinamen, gefolgt vom Namen der Anwendung, an. Um den Text der Titelzeile zu hören oder festzustellen, in welcher Anwendung Sie sich befinden, drücken Sie JAWS TASTE+T.

Menüleiste

Die Menüleiste wird mit ALT aktiviert. Sie enthält eine Reihe von Begriffen, die, sobald sie aktiviert werden, ein Menü ausklappen. In der Menüleiste finden Sie die Befehle zum Verändern Ihrer Dateien. Die Menüleiste von HJPad enthält die Begriffe Datei, Bearbeiten, Ansicht, Einfügen, Format, Extras, und Hilfe. Machen Sie sich mit den Befehlen in den HJPad Menüs vertraut. HJPad ist eine gute Umgebung zum Üben, und es werden viele Menübefehle anderer verbreiteter Anwendungen eingesetzt. Öffnen Sie einen Eintrag der Menüleiste mit EINGABE, und benutzen Sie PFEIL RAUF und RUNTER zum Navigieren in den Menüs.

Hinweis: Die Elemente der Menüleiste und die Menüeinträge können je nach Anwendung unterschiedlich sein.

Symbolleisten

Die Symbolleisten befinden sich direkt unter der Menüleiste. Es können je nach Anwendung eine oder beliebig viele Symbolleisten vorhanden sein. In HJPad gibt es zwei Symbolleisten: Standard und Format. Die meisten Anwendungen verfügen standardmäßig über eine Standard- und eine Format-Symbolleiste. Symbolleisten bestehen aus grafischen Schaltern, die der sehende Anwender mit der Maus anklickt. Die meisten Schalter der Symbolleisten stehen für Funktionen, die Sie auch in den Menüs finden.

Dokumenteingabefeld

Das Dokumenteingabefeld befindet sich direkt unter der Menüleiste und den Symbolleisten. Dieses Feld nimmt den größten Bereich des Fensters ein. Hier erledigen Sie den Hauptteil Ihrer Arbeit wie zum Beispiel das Verfassen eines Dokuments oder die Bearbeitung einer Tabelle.

Statuszeile

Die Statuszeile befindet sich unterhalb des Dokumentfensters und kann mit herkömmlichen Windows Befehlen nicht aktiviert werden. Die Statuszeile liefert nützliche Informationen über Ihr Dokument. In HJPad zeigt sie Erklärungen über das von Ihnen ausgewählte Menüelement an. In Microsoft Word ist die Statuszeile komplizierter aufgebaut. Dort enthält sie Informationen über die Seitenzahl, Abschnittsnummer, die Position der Schreibmarke und vieles andere. Um die Statuszeile mit JAWS zu lesen, drücken Sie JAWS TASTE+SEITE RUNTER.

Steuerungsmenü

Das Steuerungsmenü wird dazu verwendet, ein Fenster wiederherzustellen, es zu verschieben, die Größe zu verändern, es zu minimieren, zu maximieren und zu schließen. Das Steuerungsmenü ist meist unsichtbar, Sie können es jedoch mit ALT+LEERTASTE öffnen. Gehen Sie dann mit PFEIL RAUF und RUNTER durch die Menüeinträge.

Den Windows Explorer verwenden

Den Windows Explorer verwenden

Ordneroptionen

Bevor Sie den Windows Explorer verwenden, sollten Sie einige Änderungen an Ihren Ordneroptionen vornehmen. Sie finden die Ordneroptionen unter Start, Einstellungen, Systemsteuerung, Ordneroptionen. Es erscheint eine Gruppe von Auswahlschaltern. Drücken Sie EINGABE, um den Schalter zu aktivieren. Drücken Sie UMSCHALT+TAB, um sich durch die Gruppen von Auswahlschaltern zu bewegen, die jetzt erscheinen. Um die Zusammenarbeit von JAWS und dem Windows Explorer zu optimieren, sollten Sie die folgenden Elemente auswählen:

  • Öffnen durch Doppelklick (Markieren durch einfachen Klick).
  • Jeden Ordner im selben Fenster öffnen.
  • Herkömmliche Windows-Ordner verwenden.

Windows Explorer

Der Windows Explorer ist Bestandteil des Betriebssystems und zeigt in hierarchischer Ansicht sowohl die Ordner und Dateien des Computers, als auch die Ordner und Dateien, die sich in einem ausgewählten Ordner befinden, an. Mit dem Windows Explorer können Sie Ihre Dateiverwaltung leicht durchführen. Er ist aufgeteilt in eine große Strukturansicht auf der linken und eine Listenansicht auf der rechten Seite. Die Listenansicht rechts zeigt den Inhalt der in der linken Strukturansicht gewählten Ordner. Sie können im Windows Explorer wie in jeder anderen Strukturansicht oder Listenansicht navigieren.

Der Windows Explorer wird in der Regel dazu verwendet, Dateien zu verschieben oder zu löschen. Verwenden Sie die Windows Standardtastenkombinationen zum Ausschneiden, Kopieren und Einfügen. Sie können mehrere Elemente gleichzeitig kopieren oder löschen, indem Sie sie zusammen auswählen.

Wechseln Sie zunächst mit TAB zur Listenansicht.

Wenn Sie zusammenhängende Elemente auswählen möchten, drücken und halten Sie die UMSCHALT-Taste, um dann mit PFEIL RUNTER die gewünschten Elemente zu markieren.

Um Elemente auszuwählen, die nicht untereinander stehen, halten Sie die STRG-Taste gedrückt und bewegen sich mit PFEIL RUNTER durch die Liste. Markieren Sie die gewünschten Elemente mit der LEERTASTE. Halten Sie die STRG Taste die ganze Zeit über gedrückt.

Eine Auswahl kann jederzeit durch Drücken einer PFEILTASTE aufgehoben werden.

Dateien und Ordner mit dem Windows Explorer verwalten

Sie können den Inhalt Ihres Computers verwalten. Sie können Dateien kopieren oder ausschneiden und sie an einem anderen Ort einfügen. Mit dem Windows Explorer können Sie auch Dateien von Ihrem Computer löschen. Um diese Befehle kennenzulernen, lesen Sie bitte Tastenkombinationen zum Kopieren, Ausschneiden und Einfügen.

Windows Explorer aktualisieren

Es kommt vor, dass der Windows Explorer Ihre Änderungen nicht sofort anzeigt, nachdem Sie Dateien kopiert oder verschoben haben. Wenn Sie zum Beispiel eine Datei gelöscht haben, JAWS diese aber immer noch vorliest, drücken Sie STRG+F5, um die Bildschirmanzeige zu aktualisieren.

Spalten ausdehnen

Es werden immer dann Elemente unvollständig angezeigt, wenn die Spalte nicht breit genug ist, um das gesamte Wort oder den gesamten Ausdruck darzustellen. Von daher liest JAWS nur das, was erscheint, und dabei kann es sich nur um einen Teil des Wortes handeln. Es kann sein, dass Sie nicht genügend Informationen hören, um herauszufinden, was eigentlich angesagt werden soll. Um die Spalte so auszudehnen, dass der gesamte Text sichtbar wird, drücken Sie STRG+NUM PLUS.

Dateien umbenennen

Um einen Dateinamen zu ändern, wählen Sie die gewünschte Datei aus und drücken F2. Sie können nun den neuen Dateinamen eingeben und mit EINGABE die Änderung abschließen. Einige Dateien kann man nicht umbenennen, andere benötigen spezielle Dateinamenserweiterungen. Windows warnt Sie in so einem Fall und gibt Ihnen die Möglichkeit, den Dateinamen in einen gültigen Namen zu ändern.

Dateien suchen

Wenn Sie eine Datei oder Anwendung auf Ihrem Computer nicht finden können, verwenden Sie die Suchfunktion, um die Datei aufzufinden. Drücken Sie F3 oder STRG+F, um das Dialogfeld Suchen zu öffnen. Geben Sie den Dateinamen teilweise oder ganz ein, und füllen Sie die anderen Felder des Dialogfeldes nach Bedarf aus. Wenn Sie den Dateinamen nicht kennen, verwenden Sie die die Suchoptionen Datum und Erweiterte Optionen in diesem Dialogfeld. Drücken Sie STRG+TAB, um auf die anderen Registerkarten zu springen. Drücken Sie den Schalter Jetzt suchen, so dass Windows alle Dateien sucht, die Ihren angegebenen Suchkriterien entsprechen. Sobald Sie mit TAB die gewünschte Datei in der Liste gefunden haben, können Sie mit EINGABE diese sofort öffnen.

Windows mit JAWS verwenden

Sobald Sie über die nötigen Grundkenntnisse verfügen, werden Sie Windows mit JAWS schnell und einfach bedienen können. In diesem Kapitel erfahren Sie alles über die Windows Bedienung mit JAWS. In diesem Abschnitt lernen Sie die Windows Umgebung kennen, Sie erfahren Einzelheiten über weitergehende Windows Komponenten und Sie sind am Ende in der Lage, Windows zu bedienen, um die Produktivität und den Spaß bei der Arbeit am Computer zu erhöhen.

In den vergangenen Jahren hat JAWS immer mit den unterschiedlichen Windows Versionen Schritt gehalten. All diese Windows Versionen sind miteinander vergleichbar. Die Anwendungen ähneln einander, alle Versionen verfügen über ein Startmenü und eine Systemsteuerung. Auch den Windows Explorer finden Sie in sämtlichen Windows Versionen wieder. Die Dialogfelder mit den spezifischen Einstellungsmöglichkeiten unterscheiden sich nur unwesentlich. Wenn wir Sie also bitten, eine Einstellung in einem bestimmten Windows Bereich zu ändern, kann es durchaus sein, dass die von uns erwähnte Einstellung nicht vorhanden ist, aber Sie werden immer eine Variation dieser Einstellung in der gleichen Umgebung finden.
