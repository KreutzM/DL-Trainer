# JAWS DE Raw Sources

- Kanonischer Rohdatenpfad für die aktuell verwendeten JAWS-DE-Hilfe-HTMLs ist `data/raw/JAWS/DE/Converted-Help-Files/`.
- Die HTML-Dateien in diesem Pfad sind importierte bzw. bereits konvertierte Help-HTMLs und nicht die finale normalisierte RAG-Basis.
- Weitere Herkunfts- und Statusangaben stehen in `data/raw/JAWS/DE/Converted-Help-Files/collection_manifest.json`.
- Die nächste Pipeline-Stufe soll diese Rohbasis nach `data/normalized/` in Markdown plus Metadaten überführen.
