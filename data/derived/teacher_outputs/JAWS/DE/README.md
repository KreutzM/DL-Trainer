# JAWS-DE Teacher Output Seeds

Dieses Verzeichnis enthält die erste kleine Supportdaten-Baseline zwischen Chunk-Korpus und späterem Review/Fine-Tuning.

- `seed_generation_jobs.jsonl`: deterministische Teacher-Jobs oder Dry-Run-Inputs
- `seed_sft_candidates.jsonl`: kleine SFT-Draftmenge
- `seed_eval_cases.jsonl`: kleines disjunktes Eval-/Holdout-Set

Die Seed-Daten sind absichtlich klein und reviewbar. Sie ersetzen keinen späteren menschlich geprüften Gold-Datensatz.
